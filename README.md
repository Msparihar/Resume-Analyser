# Resume-Analyser

## Step - 1
### Activating Environment
```
.\venv\Scripts\activate
```
## Step - 2
### Installing Requirements
```
pip install -r requirements.txt
```
## Step - 3
### Installing Spacy model
```
python -m spacy download en_core_web_sm
```
## Step - 4
### Running app
``` 
python app.py
```
